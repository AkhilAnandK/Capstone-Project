1. Install Python

First, you need to install Python in order to run the capstone.py file.

2. Install Required Libraries

Once Python is installed, you need to install the required libraries such as NumPy, pandas, TensorFlow, transformers, and Keras.

3. Place CSV Files

Next, place the uploaded CSV files(train_main, dev_main, test_main) in the folder where the Python is installed.

4. Place the capstone.py file

After that, put the capstone.py file in the same destination mentioned above.

5. Run the Code

Finally, open the capstone.py file using Python idle and run the code.

6. The trained model will be saved with the name Capstone_model.h5.

7. The predictions of the model will be saved in the python folder with the name predictions.xlsx.
